
# 🌐 Netzwerk-Grundlagen

# Einführung
Netzwerktechnik beschäftigt sich mit der Verbindung von Computern und Geräten, damit diese miteinander kommunizieren können.


# Was ist ein Netzwerk?
Ein Netzwerk ist eine Verbindung von mehreren Geräten, die Daten austauschen.

 Beispiele:
- Heimnetzwerk (WLAN zuhause)
- Firmennetzwerk
- Internet (größtes Netzwerk)

#Netzwerktypen

- LAN (Local Area Network):** kleines Netzwerk (zuhause, Büro)
- WAN (Wide Area Network):** großes Netzwerk (Internet)
- WLAN:** drahtloses Netzwerk

---

# Wichtige Netzwerkgeräte

Router
- Verbindet dein Netzwerk mit dem Internet

##Switch
- Verbindet Geräte im lokalen Netzwerk

### Access Point
- Stellt WLAN bereit



## IP-Adresse

Eine IP-Adresse ist eine eindeutige Nummer für jedes Gerät im Netzwerk.

📌 Beispiel:
192.168.1.1



## DNS (Domain Name System)

DNS übersetzt Domainnamen in IP-Adressen.

📌 Beispiel:
google.com → 142.xxx.xxx.xxx


## Ports & Protokolle

Ports sind „Türen“, über die Daten übertragen werden.

📌 Beispiele:
- 80 → HTTP  
- 443 → HTTPS  
- 53 → DNS  
## TCP und UDP

# TCP
- zuverlässig  
- langsamer  

## UDP
- schneller  
- weniger zuverlässig  
# OSI-Modell

Das OSI-Modell besteht aus 7 Schichten und beschreibt die Kommunikation im Netzwerk.

Wichtige Schichten:
- Anwendung (HTTP)  
- Transport (TCP/UDP)  
- Netzwerk (IP)  
# Warum Netzwerke wichtig sind

- Kommunikation zwischen Geräten  
- Internetzugang  
- Datenaustausch  

#  Was passiert beim Aufruf einer Website?

##  Ablauf

1. URL wird eingegeben
2. DNS-Server wird gefragt
3. IP-Adresse wird ermittelt
4. Verbindung über TCP wird aufgebaut
5. HTTPS sorgt für Sicherheit
6. Anfrage wird gesendet (HTTP GET)
7. Server sendet Daten zurück
8. Browser zeigt die Website an

## Fazit
Der Aufruf einer Website besteht aus mehreren Schritten wie DNS, TCP, HTTP und sorgt für die Darstellung einer Webseite im Browser.
